La estructura de html consta de: el tipo de documento, cabeza de pagina y cuerpo de pagina.
Tipo de documento(!DOCTYPE html): En esta primera seccion se inserta el tipo de documento que es el html, en caso de especificar el tipo documento, el navegador entrara en modo recompatibilidad.
Cabeza de pagina (head): es la cabezera del html, se inserta el titulo del html, icono etc. Se incluyen ciertas etiquetas de metadatos, es decir, unas etiquetas que establecen ciertos datos que no tienen que verse necesariamente de forma visual.
Cuerpo de pagina (body): es el cuerpo de html, aqui se incluye todos los elementos de la pagina web .
